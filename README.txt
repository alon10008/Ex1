מימוש ממשק weighted_graph במחלקת WGraph_DS מכילה: מחלקה פנימית NodeInfo 
שממשת את ממשק node_info וhashMap ששומר את קודקודי הגרף לפי הkey.

מחלקת NodeInfo מכילה מספר שלם key שמשמש כמזהה לכל קודקוד, tag שישמש לאלגוריתם חיפוש מסלול
ושתי hashMap :
אחד שומר את השכנים של הקודקוד לפי key, והשני שומר את המשקל של הצלעות של השכנים
לפי הkey.

לכל הוספה או הסרה של צלע בגרף צריך להוסיף/להוריד את הקודקוד בשתי הhashMap בשתי הקודקודים.

מימוש ממשק weighted_graph_algorithms במחלקת WGraph_algo מכיל אובייקט WGraph_DS שעליו יעבדו האלגוריתמים.
--shortestPathDis  ממומש באמצעות dijkstra algorithm בשימוש בתור עדיפויות שמשווה לפי הtag של כל קודקוד.

--isConnected משתמש בdijkstra algorithm ובסוף בודק אם קיים קודקוד שאין לו הורה. אם קיים קודקוד כזה
סימן שהגרף לא קשיר מכיוון שאין מסלול מקודקוד מקור כלשהו אל הקודקוד הזה.
